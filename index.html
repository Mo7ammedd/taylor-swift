TAYLOR SWIFT API

Open Source API for Taylor Swift lyrics

API End-Points

  • GET RANDOM LYRICS: api/v1/quote
    • QUERY PARAMETERS:
    • GET RANDOM LYRICS FROM AN ALBUM: api/v1/quote/:album
    • GET RANDOM LYRICS FROM A SONG: api/v1/quote/:song
  • GET ALL LYRICS MATCHING THE FILTERS:
    • FILTER BY SONG: get-all?song=<song>
    • FILTER BY ALBUM: get-all?album=<album>
  • GET RANDOM SONG : /api/v1/rsong
  • add taylor swift lyrics to the API:
    • open postman and use post: api/v1/quote
    • with this format
      { "quote":"test",
      "song":"test",
      "album":"test" }
  • Generate quiz questions :
    • api/v1/quote/quiz/question
    • { "question": "the question will be here",
      "answers": [ "Song A", "Song B", "Song C", "Song D" ] }

      • question: The quote as a string.
      answers: An array of four strings, where the first element is the correct answer and the remaining three are incorrect answers.
